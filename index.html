Recent Work

EMSwift

EMSwift was created to help first responders enter vital information. Currently the programs being used are too complicated in such a time sensitive environment. This program was created to facilitate the ease of use for all levels of care and potentially for much more than medical applications alone.

UAC Tracker 3000

This program is created for a workforce to remove redundancies from form building. Initially used to create accountability while tracking un-accompanied children going through the migration system. This format can be used in a variety of applications to ease workflow.

5 O'clock Somewhere

We have all wanted a drink after work. This app will allow you to take ingredients you have on hand and see what you can whip up. It also has libraries and suggestions you can take in with you to your next stop at the local liquor store.

Tech Helper

Using recursive methods this is a tool that can be used by a service technician for air conditioning to narrow down potential issues with a non-working AC or heating unit. Unfortunately, it is not complete as it was my final project and I was working alone, the cooling side is half complete presently.
